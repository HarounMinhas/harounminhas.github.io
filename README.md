# Haroun Minhas Portfolio

Portfolio repository for GitHub Pages.

## Live website

- Portfolio: [https://harounminhas.github.io/](https://harounminhas.github.io/)

## What is in this repo

- `portfolio-index.html`: main portfolio page
- Project folders with standalone apps and demos
- `MusicDiscovery/`: deployed output from `HarounMinhas/MusicDiscovery`

Do not edit files inside `MusicDiscovery/` in this repository. They are overwritten by the external deployment flow.

## Local use

- Open `portfolio-index.html` directly for the portfolio page
- Use project-specific README files inside subfolders for local setup of those projects
