# Zen Writer

Minimalist distraction-free writing environment focused on productivity and flow state.

## Features

- **Distraction-Free Mode**: Full-screen writing interface with minimal UI
- **Auto-Save**: Continuous automatic saving to browser storage
- **Word Count**: Real-time word and character tracking
- **Focus Timer**: Pomodoro-style timer integration
- **Markdown Support**: Write in Markdown with live preview
- **Theme Selection**: Light, dark, and sepia modes
- **Export Options**: Save as TXT, Markdown, or HTML
- **Keyboard Shortcuts**: Efficient navigation and formatting

## Technology Stack

- **Single HTML File**: Self-contained application
- **Vanilla JavaScript**: No external dependencies
- **LocalStorage**: Persistent draft storage
- **CSS3**: Responsive typography and theming

## Usage

Open `index.html` directly in a web browser. No build process or server required.

## Keyboard Shortcuts

- `Ctrl/Cmd + S`: Save current document
- `Ctrl/Cmd + E`: Toggle Markdown preview
- `Ctrl/Cmd + F`: Toggle focus mode
- `F11`: Toggle fullscreen
- `Ctrl/Cmd + ,`: Open settings

## Features in Detail

### Auto-Save
Documents are automatically saved to browser localStorage every few seconds. Recovery is automatic on page reload.

### Focus Timer
Optional Pomodoro timer helps maintain writing sessions with configurable work/break intervals.

### Markdown Preview
Real-time Markdown rendering allows you to write structured content and preview formatted output side-by-side.

### Export Formats
- Plain text (.txt)
- Markdown (.md)
- HTML with styling

## Browser Support

Works in all modern browsers with localStorage support. Best experienced in latest Chrome, Firefox, Safari, or Edge.

## Privacy

All data is stored locally in your browser. No data is sent to external servers. Clear browser data to remove saved drafts.
